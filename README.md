# Wearable-Project

Featuring EMG.

1. Complete Eagle File
	a. 11 input pins (+ 3 internal battery pins + 2 internal output pins)
	b. 1 output pin
	c. 3 battery pins
	d. 1 Arduino ground pin
	e. 1 patient ground pin
2. Order PCB
3. Combine components
	a. PCB
	b. Bluno Beetle (Jerry’s app)
	c. Batteries
	d. Electrodes
4. Attach the “nervous system” to the “skin”
5. Collect more data (Jerry’s different classes)
6. Pass data through seglearn
	a. Option 1: get seglearn working on a phone
	b. Option 2: get the data off the phone and run seglearn on a computer
7. Utilize results in the app
